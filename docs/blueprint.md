# **App Name**: RutaLegal Colombia

## Core Features:

- Simulador de Examen Completo: Una interfaz de prueba interactiva que permite a los usuarios completar el cuestionario de 25 preguntas sobre leyes de tránsito colombianas con un temporizador opcional.
- IA Asesor Legal de Tránsito: Un motor de IA que utiliza el Código Nacional de Tránsito como herramienta para explicar el razonamiento detrás de cada respuesta correcta.
- Calculadora de Calificación RUNT: Lógica para procesar las respuestas y emitir un veredicto de 'Aprobado' o 'Reprobado' basado en los estándares oficiales de las escuelas de conducción.
- Retroalimentación de Seguridad Pasiva y Activa: Información detallada sobre mecánica básica y elementos de seguridad que se activa tras completar los bloques correspondientes.
- Modo de Repaso Inteligente: Función impulsada por IA que identifica tus debilidades y genera preguntas personalizadas sobre las áreas donde tuviste más errores.
- Exportación de Reporte de Resultados: Genera una versión en PDF de los resultados para que el estudiante pueda revisarlos offline antes de su cita oficial.

## Style Guidelines:

- Primary color: Deep Midnight Blue (#1A3A5A) representing authority and the order of transit regulations.
- Background color: Clean Steel (#F1F4F6), a heavily desaturated blue hue providing high focus and readability.
- Accent color: Cobalt Violet (#4A3A8A), providing a sophisticated, analogous contrast for calls to action.
- Font pairing: 'Space Grotesk' for modern, techy headlines and 'PT Sans' for warm, highly readable body text and explanations.
- Outline-style icons inspired by Colombian transit signals (diamond, circle, triangle shapes) used for section indicators.
- Structured single-column layout optimized for vertical scrolling, using card-based containers with soft elevation.
- Smooth slide-in transitions when the AI reveals feedback, and a progress bar that updates as the user completes questions.
